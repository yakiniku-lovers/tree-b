みんなでたのしくうんぱっぱのぷんぷん

## Requirements
* Python 3
* MeCab
* `pip install mecab-python3`

## Installation

1. Clone this repository
```shell
git clone
```

2. Init submodule
```shell
git submodule init
```

3. Update submodule
```shell
git submodule update
```

## Usage
```shell
ruby main.rb <tweets.json>
```
